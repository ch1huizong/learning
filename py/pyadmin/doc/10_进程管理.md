# 第十章 进程管理/并发/调度

### 一. 进程包
	
1. subprocess

    * call 只执行系统命令/禁止输出/退出码	

    * Popen	可以输入/输出重定向
        stdout=subprocess.PIPE 选项意思过滤到标准输出


2. supervisor 管理"常驻"系统进程(挺有用)

    * supervisord 守护进程, 自定义配置文件

    * supervisorctl	客户端


3. screen 进程分离


4. threading 处理并发

    * Thread

    * Timer	定时线程

5. multiprocessing
